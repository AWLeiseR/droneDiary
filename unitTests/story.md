# Drone Diary

### Goal

-   Keep record of the conditions of each drone flight

### Creation Cases

#### User case

-   A valid user must have:
    -   id: string
    -   name: string
    -   email: string

#### Location case

-   A valid location must have:
    -   id: string
    -   name: string
    -   latitude: number
    -   longitude: number

#### Note case

-   A valid note must have:
    -   id: string
    -   date: string
    -   startTime: string
    -   endTime: string
    -   areaSize: number
    -   picTotal: number
    -   descriptionFlight: string
